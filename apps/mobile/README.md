# Mobile App Scaffold (Android + iOS)

This directory is reserved for the NeuroLift cross-platform mobile app.

## Target stack
- React Native + Expo
- TypeScript
- Shared API contracts from `packages/shared`

## Responsibilities
- Daily check-ins
- Coaching prompts and interventions
- Lightweight scenario playback and reflection
- Push notifications and reminders

## Next implementation step
Initialize with Expo:

```bash
npx create-expo-app apps/mobile --template
```
