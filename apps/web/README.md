# Web App Scaffold

This directory is reserved for the NeuroLift full-stack web application.

## Target stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Auth + role-aware dashboards

## Responsibilities
- Simulation session creation and management
- Progress analytics and reporting
- Admin tools for scenario libraries and Avatar/Aide configs
- Billing/subscription UI (future)

## Next implementation step
Initialize the web app:

```bash
npx create-next-app@latest apps/web --ts --tailwind --eslint --app --src-dir
```
