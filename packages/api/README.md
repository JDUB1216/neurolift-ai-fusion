# API Service Scaffold

This package will host the full-stack backend used by web and mobile clients.

## Target stack
- Node.js + TypeScript
- Fastify or NestJS
- PostgreSQL/Supabase
- Redis (jobs/session state cache)

## Core domains
- Auth & user profiles
- Simulation session orchestration
- Avatar/Aide/Advocate state persistence
- Analytics and reporting endpoints
- Notification scheduling hooks

Set up TypeScript API service scaffold and connect to existing Supabase migrations.
